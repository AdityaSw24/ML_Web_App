# Diabetes Predictor App
 Try the deployed model here -> [https://mlwebapp-99mtpspwpwvklvkyabbb4f.streamlit.app/]
- It is a Trained Machine Learning model deployed as Streamlit Application that predicts a person having diabetes or not based on the input medical data
- Data we input:
  - Age
  - Number of pregnancies 
  - Glucose level
  - Blood Pressure value 
  - Skin Thickness value
  - Insulin Level
  - Body-Mass Index
  - Diabetes Pedigree Function value
